# Gestão de Estoque - Casa da Itália

Um sistema web simples para gestão de estoque e receitas de restaurante, desenvolvido para o restaurante Casa da Itália.

## 📋 Descrição

Este projeto é uma aplicação web que permite gerenciar o estoque de insumos, controlar receitas e monitorar o desempenho financeiro do restaurante. Utiliza armazenamento local do navegador para persistência de dados.

## ✨ Funcionalidades

- **Login Seguro**: Acesso protegido com usuário e senha
- **Dashboard**: Visão geral com estatísticas e gráfico de distribuição financeira
- **Gestão de Estoque**: 
  - Adicionar novos insumos
  - Controlar quantidades mínimas
  - Movimentar estoque (+/-)
  - Alertas para itens abaixo do mínimo
- **Gestão de Receitas**: 
  - Criar fichas técnicas de pratos
  - Calcular margem de lucro
- **Backup**: Exportar dados em formato JSON
- **Interface Responsiva**: Design moderno com sidebar e painéis de vidro

## 🚀 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/gestao-estoque-restaurante.git
   cd gestao-estoque-restaurante
   ```

2. **Abra o arquivo**:
   - Abra `index.html` em qualquer navegador moderno

3. **Login**:

   - Senha: `1234`

4. **Navegação**:
   - Use a sidebar para alternar entre Dashboard, Estoque e Receitas
